# NumPy Analyzer

## 📌 Project Overview
NumPy Analyzer is a menu-driven Python application developed using **Object-Oriented Programming (OOP)** concepts and the **NumPy** library. It allows users to create and manipulate NumPy arrays while performing various mathematical, statistical, and analytical operations through an interactive console interface.

---

## 🚀 Features

### 1. Array Creation
- Create 1D arrays
- Create 2D arrays
- Create 3D arrays

### 2. Indexing & Slicing
- Access array elements using indexing
- Slice arrays using row and column ranges

### 3. Mathematical Operations
- Addition
- Subtraction
- Multiplication
- Division
- Dot Product
- Matrix Multiplication

### 4. Combining & Splitting Arrays
- Combine multiple arrays
- Split arrays into equal parts

### 5. Search, Sort & Filter
- Search specific values
- Sort arrays in ascending or descending order
- Filter values using conditions
  - Greater than (>)
  - Less than (<)
  - Greater than or equal (>=)
  - Less than or equal (<=)
  - Equal to (==)

### 6. Statistical Analysis
- Sum
- Mean
- Median
- Standard Deviation
- Variance
- Minimum & Maximum
- Percentile
- Correlation Coefficient
